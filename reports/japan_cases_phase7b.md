## 1. マサキング SHIBUYA POKER FES コール後フォールド事件

**Case ID**: `JP-01-masaking`

**期待される判断**: 両プレイヤーがハンドを公開した時点で showdown 成立 + 「コルコルコル」発声がコール宣言として有効 → call binding 確定。後付けの fold は認められない。チップを支払う義務あり。Rule 41 (Methods of Calling) + Rule 16 (Face Up for All-Ins)。フロアは事実認定義務を最大限果たすべきだった。

**期待される引用ルール**: `Rule-16, Rule-41`

### 入力

```
状況: SHIBUYA POKER FES の River でプレイヤー① が KT で all-in。プレイヤー② が「コルコルコル」と発声しながら AT を見せた。ディーラーが「コールと言わずにショーしたのでコール扱いにならない」と判定。フロアが呼ばれ、両プレイヤーがハンドを開いたままプレイヤー② のアクション継続を指示。プレイヤー② はその後 fold を宣言し、チップを払わずハンド終了。生配信で「コルコルコル」発声の音声証拠が残っている。
```

### AI 出力

```
【推奨判断】
プレイヤー②の「コルコルコル」発声は明確なコール宣言であり、all-in額を支払う義務が発生する。

【適用ルール】
- **Rule-40A**: Methods of Betting: Verbal and Chips（主要）
- **Rule-41A**: Methods of Calling（補助）
- **Rule-16**: Face Up for All-Ins（補助・カード開示済みの扱い）

【根拠】
Rule 40A は「bets are by verbal declaration and/or pushing out chips. If a player does both, whichever is first defines the bet」と規定しており、Rule 41A は「standard and acceptable forms of calling include: A) saying "call"」と明示している。「コルコルコル」は口語的表現だが、音声証拠により「call」の発音意図が明確であり、コール宣言として有効である。チップを押し出す前に口頭宣言が先行したため、宣言が優先される。

ディーラーの当初判定「コールと言わずにショー」は誤りである。プレイヤー②がカードを開示した行為は Rule 16 の文脈では all-in に対する showdown 行為と解釈でき、fold 意思表示とは矛盾する。

【裁量余地】
ただし、現場 TD は「コルコルコル」の発音明瞭度・文脈・他プレイヤーの認識により、宣言の有効性を再評価することができる。

【ペナルティ】
- プレイヤー②: コール額支払い義務（Rule 40A 適用）
- プレイヤー①: 既に all-in 済みのため追加アクションなし
- ディーラー/フロア: 手順ミス（教育的指導対象、ただしプレイヤーへのペナルティではない）

【代替案】
- A: 音声証拠が不明瞭な場合、「コールの意思表示が不明確」として Rule 1 の裁量で fold 容認（条件: 複数証人が「コール」と認識していない）

【確認事項】
- 配信音声の明瞭度と「コール」発音の確実性
- プレイヤー①がプレイヤー②のカード開示を「コール」と認識したか
- ハウスルールで口語的コール表現の扱いが規定されているか

【確信度】high

## 2. TPC 大阪 ファイナルテーブル 3 人組打ち優勝事件

**Case ID**: `JP-02-tpc-collusion`

**期待される判断**: Rule 67 (One Player to a Hand) 違反 + Rule 71 ペナルティ階層で DQ レベル。ただし疑惑段階では即 DQ せず、まず別卓分離 + 監視強化、RFID 配信ログを精査して確証を得てから DQ + chip forfeit。確定後は賞金プール返還 or 残プレイヤーへの再分配を TD 裁量で。

**期待される引用ルール**: `Rule-1, Rule-67, Rule-71`

### 入力

```
状況: TPC（Top of Poker Championship）大阪のファイナルテーブルで、特定の 3 人のプレイヤーが事前打ち合わせの疑い。お互いを避け、特定 1 人にチップを集中させる動きが繰り返された。結果として 1 位・2 位・3 位を独占。RFID 配信映像で組打ちが疑われ、SNS で大炎上。他プレイヤーから複数の苦情申告。TD として疑惑段階でどう対応するか、確定後はどう処分するか。
```

### AI 出力

```
【推奨判断】
疑惑段階では即DQせず、3名を別卓分離+個別聴取+RFID映像精査を実施し、証拠が揃った時点でDQ+賞金没収+残プレイヤーへの再分配を検討する。

【適用ルール】
- **Rule-69**: Ethical Play（主要）
- Rule-67: No Disclosure, One Player to a Hand（補助）
- Rule-71: Warnings, Penalties, and Disqualification（補助）

【根拠】
Rule 69は「Chip dumping and other forms of collusion will result in disqualification」と明記しており、組打ち確定時はDQが標準処分。ただし疑惑段階での即処分は訴訟リスクを伴うため、以下の段階を踏む必要がある：

1. **観察・分離フェーズ**：3名を異なるテーブルに分離し、独立したディーラー・フロアスタッフで監視
2. **証拠収集**：RFID映像の全ハンド精査、他プレイヤーからの具体的証言録取、ベット額・タイミング・HUD統計の異常パターン分析
3. **個別聴取**：各プレイヤーから個別に事情聴取（相互矛盾点を確認）
4. **確定判断**：複数証拠が一致し、合理的疑いを超えた時点でDQ決定

Rule 67の「One-player-to-a-hand」違反（戦略共有）とRule 71の「cheating」に該当する場合、段階的ペナルティではなく即DQが正当化される。

【裁量余地】
ただし、現場TDは証拠の強度・他プレイヤーの証言信頼性・映像解析結果により、DQではなく賞金減額や次回出場停止などの代替処分を選択することができる。

【ペナルティ】
- **疑惑段階**：なし（分離監視のみ）
- **確定後**：DQ + 賞金全額没収 + 残プレイヤーへの賞金プール再分配 + TPC出場永久停止検討

【代替案】
- A: 証拠が状況証拠のみの場合、DQではなく「賞金50%減額+次回大会出場停止1年」（条件：映像に決定的証拠なし、ただし統計的異常は明白）
- B: 1名のみ主導と判明した場合、主導者のみDQ、他2名は厳重注意+賞金30%減額（条件：個別聴取で主従関係が明確）

【確認事項】
- RFID映像で以下の異常パターンが3回以上記録されているか：①明らかなナッツで降りる ②BBvsチップリーダーで異常に弱いディフェンス ③3名以外が参加するポットでの協調レイズ
- 他プレイヤーからの苦情が具体的ハンド番号・時刻付きで3件以上あるか
- SNS炎上が事実誤認に基づく可能性はないか（第三者による冷静な映像検証が必要）

【確信度】high

```

## 3. JOPT 公式ツイートで個人情報流出

```

## 4. ディーラーがアクティブハンドを誤って回収

**Case ID**: `JP-04-dealer-collected-active`

**期待される判断**: Rule 65A (Players must protect their hands at all times) により、原則としてプレイヤー側の保護義務違反 → ハンド dead。カードプロテクター（チップ等）を置いていなかった場合は救済不可。ただし TD は当時の状況・常連性・ディーラー過失の程度から Rule 1 でベット返却（pot は失う）等の折衷案も検討可能。

**期待される引用ルール**: `Rule-1, Rule-14, Rule-65`

### 入力

```
状況: NLHE、Day 1 後半、Blinds 2,000-4,000。プレイヤー A が River でハンドを伏せて考えていた（カードプロテクターなし）。ディーラーが誤って A のカードを回収し、マック山に混ぜてしまった。A は「俺のハンドだ、まだ降りてない」と主張。ディーラーは「もう特定できない」と返答。ポットは 80,000、A のスタックは 120,000、まだリバーアクション前。フロアに判断を求める。
```

### AI 出力

```
【推奨判断】
A のハンドは死亡、ポットへの権利を失う。未コールベット（A が前ラウンドまでに投入した分）は返金されない。

【適用ルール】
- **Rule 65A**: Accidentally Killed / Fouled / Exposed Hands（主要）
- Rule 13B: Tabling Cards and Killing Winning Hand（補助・プロテクト義務）

【根拠】
Rule 65A は「Players must protect their hands at all times」と明記し、ディーラーミスでハンドが回収され100%特定不能な場合、プレイヤーは救済を受けられず、コール済みベットの返金もないと規定している。A はカードプロテクターを使わず、River でハンドを伏せたままディーラーの誤回収を防げなかった。TDA はこの状況を「プレイヤー側の保護義務違反」と扱う。

【裁量余地】
ただし、現場 TD は監視映像・ディーラー証言・周囲プレイヤー証言によりハンドが 100% 特定できると判断すれば、Rule 65B に基づきハンドを復活させることができる。

【ペナルティ】
プレイヤー A へのペナルティなし（ディーラーミスが契機のため）。ディーラーへの再教育推奨。

【確認事項】
- 監視映像で A のホールカードが映っているか
- テーブル他プレイヤーが A のカードを視認していたか
- マック山の枚数がデッキ復元で逆算可能か

【確信度】high

---
**出典**: TDA Rule 65A, 13B used by permission, Copyright 2024, https://www.pokertda.com. 最終判断は現場 TD に委ねられます。
```

## 5. 2024 ストリングベット新定義の現場混乱

**Case ID**: `JP-05-stringbet-2024`

**期待される判断**: 2024 改訂 Rule 56 で string bet の定義は「1 度ベットしたあと再びスタックに手を伸ばし追加チップをベットする複数の動作」に限定された。「上からチップをバラバラ落とす」「1 本のスタックを前に出して一部を下げる」は string bet ではない（許容）。本件のように上から落とした場合は valid bet として扱う。古い解釈で string bet 判定するのは TD 側のミス。

**期待される引用ルール**: `Rule-42, Rule-56`

### 入力

```
状況: NLHE、Day 2、Blinds 3,000-6,000。Preflop、UTG オープン 15,000。CO がチップスタック 1 本（5,000 × 6 枚）を前に押し出したが、上から数枚をバラバラと落とすような動作をした。宣言なし。総額は 30,000。ディーラーが「string bet です、コール額（15,000）のみ受け付け」と判定。プレイヤーは「2024 ルール改訂で上から落とすのは string じゃないはず」と異議。フロアの判断を求める。
```

### AI 出力

```
【推奨判断】
30,000 の raise として受理する（string bet ではない）。

【適用ルール】
- **Rule-56**: String Bets and Raises（主要）
- Rule-42: Methods of Raising（補助）
- Rule-40: Methods of Betting: Verbal and Chips（補助）

【根拠】
Rule-56 の 2024 年改訂では「String bets and raises involve multiple movements whereby a player puts out a bet then returns to their stack for more chips to add to the bet」と定義される。本ケースはスタック 1 本（30,000）を前に押し出した単一動作であり、**スタックに戻って追加チップを取る動作がない**。上から数枚を落とす動作は、push out の過程で生じた物理的な崩れであり、Rule-56 が禁止する「returns to their stack」には該当しない。Rule-42 では「pushing out the full amount in one motion」が正当な raise 方法として認められており、本件はこれに該当する。

【裁量余地】
ただし、現場 TD が「スタックを分割して 2 回に分けて押し出した」と事実認定すれば、string bet として call 扱いとする余地もある。

【ペナルティ】
なし（プレイヤーに悪意なし、ルール解釈上の争点）

【確認事項】
- CO プレイヤーがスタックを押し出した後、手を引いたか、それとも押し出し途中で崩れただけか
- ディーラー・他プレイヤーの視認情報（明確な 2 モーションだったか）
- 映像確認が可能な場合は、動作を再生して判断材料とする

【確信度】high

---
**出典**: TDA Rules 2024, Rule-56 (String Bets), Rule-42 (Methods of Raising). **最終判断は現場 TD に委ねられます。**
```

### メトリクス

- **応答時間**: 17.8s
- **コスト**: 約 3.81 円
- **確信度**: high

- **引用ルール（RAG）**: `Rule-56, Rule-42, Rule-45, Rule-40, Rule-16`
